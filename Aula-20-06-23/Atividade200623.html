Atividade - Tags de Texto

Nesta atividade, você aprenderá a utilizar tags de texto para formatar o conteúdo em uma página web

A seguir, temos um exemplo de trecho de código em HTML para enfatizar um texto.

<em>Texto enfatizado</em>
<em>Lorem ipsum dolor sit amet consectetur adipisicing elit. Sint ratione dolorum et provident quidem, nam ex unde reiciendis eum vitae fugiat error repellat facilis nisi, nostrum eaque quae voluptate recusandae?</em>

Experimente utilizar outras tags de texto, como <strong>, <u>, <sub>, <sup>, entre outras, para ver como elas alteram a aparência do texto.

Exemplo <strong> Negrito (Aparência + Mais relevante): Lorem ipsum dolor sit amet consectetur adipisicing elit. Libero molestias corrupti possimus cum facilis, sint sunt, consequatur sed temporibus sapiente tenetur quidem veritatis qui esse fugiat quos. Molestias, in aperiam?

Exemplo <u> (Sublinhado): Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quisquam assumenda temporibus mollitia labore nemo nam libero animi repellat! Molestias ullam aliquam magnam atque id minima dolorum amet nostrum corrupti soluta.

Exemplo <sub> (Subescrito): Lorem ipsum dolor sit, amet consectetur adipisicing elit. Maiores iure quis neque deserunt laborum ducimus, quidem quia facilis. Dicta possimus neque veritatis deserunt cupiditate explicabo ipsam esse quas facilis necessitatibus!

Exemplo <sup> (Sobreescrito): Lorem ipsum dolor sit amet consectetur adipisicing elit. Fuga placeat sapiente, quos tempora perferendis aspernatur delectus doloremque commodi reprehenderit omnis est quisquam quam ipsa tempore tenetur qui! Quos, officiis autem.
